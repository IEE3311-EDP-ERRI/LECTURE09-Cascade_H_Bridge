# LECTURE09-Cascade_H_Bridge
This repo contains the fundamental knowled of control and modulation of Casacde H-Bridge Converter. <br>

[This](https://github.com/IEE3311-EDP-ERRI/LECTURE09-Cascade_H_Bridge/blob/main/Thesis_Documenttion.pdf) thesis chapters allow you to understand the topology and standard modulations algorithm: Phase-shift PWM, Level-shift PWM, and Space Vector Modulation. The STATCOM operation just requieres external control of P=0 and Q according to an external reference. The floating dc-link voltage of each cell is implemented in the simulation section.
